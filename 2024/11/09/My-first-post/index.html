什么是 Markdown？

Markdown 是一种轻量级的标记语言，允许你使用简单的文本格式来创建结构化文档。它广泛用于编写文档、博客和网页。

Markdown 的基本语法

  • 标题：使用 # 表示标题。
  • 段落：直接输入文本。
  • 列表：
    • 无序列表项
    • 另一个项
  • 有序列表：
    1. 第一项
    2. 第二项
